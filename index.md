## Club Compass - Redefining Club Discovery

Club Compass is a full stack application to aid students at high schools in finding, creating, and managing on campus clubs. It is a web application that allows students to create clubs, invite other students to join, and manage club events. One of the largest problems with club discovery is that students are not able to find clubs that are relevant to their interests. Club Compass is a tool that allows students to find clubs that are relevant to their interests.

## Features

Just a basic overview of what club compass is capable of.

- Admin dashboard with full website management capabilities

  - Approve (activate) accounts
  - Edit and delete accounts
  - Create and delete clubs
  - Approve (activate) clubs
  - Edit clubs
  - Create and delete club events
  - Manage user invitations

- Full battle tested authentication system for creating and logging into accounts

  - Email verification required for all new accounts
  - Account is not activated until admin approves

- Users can...

  - Create and edit clubs
  - Manage user roles
  - Set club visibility (open, invite only, or closed)
  - Invite other users to join clubs
  - Join clubs through invitations or club page

- Clubs
  - Created and approved clubs are visible to all users through the clubs page and the clubs corresponding slug
  - A route is automatically generated when a club is approved
  - Clubs page
    - Filter by tags, search, and sort clubs
    - Join a club through the join button
  - Club page
    - View all club information and events
    - View club members and their roles
